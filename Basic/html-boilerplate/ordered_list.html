Ordered and unordered lists

Ordered lists

There are two types of lists that we can use in HTML, the first is the ordered list.

From the name, you can assume that this option follows an order when displaying the individual elements. Let's take a look at an example:

  1. Iten 1
  2. Iten 2
  3. Iten 3
  4. Iten 4

As can be seen in the list above, all elements are numbered and each additional element added to the list is given a new number.

To create an ordered list, 3 HTML tags must be used. The first is the < ol > tag, which is used to create an ordered list.

Then we use the < li > tag to create the list Iten and finally we use the < a > tag, which we use to create a link within the < li > tag.

Example of the code behind the ordered list: 


    < ol >
            < li >
                     < a > Iten 1 < / a >
            < / li >
            < li >
                     < a > Iten 2 < / a >
            < / li >
            < li >
                     < a > Iten 3 < / a >
            < / li >
            < li >
                     < a > Iten 4 < / a >
            < / li >
    < / ol >
