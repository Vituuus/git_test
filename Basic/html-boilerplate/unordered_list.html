Ordered and unordered lists

Unordered lists

Now we will talk about the unordered lists, by the name we know that this type of list does not follow a specific order.

As can be seen in the example, there is no order in the iten list, this type of list can be used to list something that does not necessarily have an order, like a supermarket list.

  • Iten 1
  • Iten 22
  • Iten 300
  • Iten 4.1

To create an unordered list, 3 HTML tags must be used, as described in the previous chapter. The first is the < ul > tag, which is used to create an unordered list.

Then we use the tag < li > tag to create the list Iten and finally we use the tag < a > tag, with which we create a link within the tag < li > tag.

Example of the code behind the unordered list: 

    < ul >
            < li >
                     < a > Iten 1 < / a >
            < / li >
            < li >
                     < a > Iten 22 < / a >
            < / li >
            < li >
                     < a > Iten 300 < / a >
            < / li >
            < li >
                     < a > Iten 4.1 < / a >
            < / li >
    < / ul >
